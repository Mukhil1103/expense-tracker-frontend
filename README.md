# 💰 Expense Tracker Frontend

A React-based frontend for managing daily expenses with JWT authentication.

🔗 Live App: https://expense-tracker-frontend-two-mauve.vercel.app

---

## 🚀 Features

- 🔐 User Login (JWT Authentication)
- ➕ Add Expenses
- ✏️ Edit Expenses
- ❌ Delete Expenses
- 📊 Expense Dashboard UI
- 🔄 Real-time updates from backend API

---

## 🛠️ Tech Stack

- React.js
- Axios
- Chart.js (for analytics)
- CSS (custom styling)

---
